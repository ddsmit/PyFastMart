# PyFastMart

## Overview

PyFastMart is a minimal cart implementation that allows unique products (one-to-one relationship with inventory and a product in Stripe, or no inventory at all for SaaS).

## Implementation Details

### Tech Stack
- **Frontend**: Vanilla HTML, CSS, JavaScript (no frameworks)
- **Backend**: FastAPI
- **Database/Products**: Stripe (source of truth for products, prices, descriptions, images)
- **CLI**: Typer-based `pfm` command
- **Configuration**: TOML for site info, `.env` for secrets

### Project Structure

```
PyFastMart/
├── pfm/                    # Python package
│   ├── __init__.py
│   ├── __main__.py        # CLI entry point (pfm manage, pfm serve)
│   ├── config.py          # Configuration management (TOML + env)
│   ├── main.py            # FastAPI application
│   └── stripe_client.py   # Stripe API wrapper
├── frontend/              # Static assets
│   ├── index.html
│   ├── styles.css
│   └── app.js
├── pyproject.toml
├── .python-version
└── config.toml            # Generated by pfm manage
```

### Configuration

The `pfm manage` command creates:
- `config.toml` - Site name, description, contact info, social links
- `.env` - Stripe API keys

### API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/site` | GET | Returns site configuration |
| `/api/products` | GET | Returns products from Stripe |
| `/api/checkout` | POST | Creates Stripe Checkout session |
| `/api/webhook` | POST | Handles Stripe webhooks |
| `/` | GET | Serves the frontend |

### CLI Commands

```bash
pfm manage    # Interactive setup wizard
pfm serve     # Start the development server
```

## Interface

- **Header**: Store title and description
- **Filters**: Search input, sort dropdown, tag filter buttons
- **Products Grid**: Responsive grid with product cards (image, name, description, price, add/remove buttons)
- **Cart**: Slide-out popover with item list, quantity controls, total, and checkout button
- **Theme Toggle**: Light/dark mode support (persisted in localStorage)
- **Footer**: Contact info and social links

## End User Deployment

Deploy to any platform that supports Python:
- Railway, Render, Fly.io, etc.
- Set environment variables for Stripe keys
- `pfm serve` to run the server
